# YouTubeDownDlp
このソフトは[yt-dlp](https://github.com/yt-dlp/yt-dlp)を使用してインターネット上から動画や音声をダウンロード出来るものです。
内部の仕組みは非表示でコマンドプロンプトを実行し、ログを取得してウィンドウに表示をしているだけです。

## セットアップ
実行ファイル直下にSystemフォルダが生成されるため、以下の実行ファイルを設置してください。  
・[yt-dlp.exe](https://github.com/yt-dlp/yt-dlp)  
・[ffmpeg.exe](https://www.ffmpeg.org/)  
・[ffprobe.exe](https://www.ffmpeg.org/)
